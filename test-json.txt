## 调试顺序建议：
# 1. 极简测试 (#5) - 验证基本功能
# 2. 基础结构测试 (#1) - 验证数组和null处理  
# 3. 元数据功能测试 (#2) - 验证标记功能
# 4. 层级不一致测试 (#7) - 验证关心的场景
# 5. 完整标记测试 (#6) - 验证所有XMind功能
# 6. 边界情况测试 (#8) - 验证异常处理
# 7. 数组根节点测试 (#4) - 验证特殊结构
# 8. 复杂嵌套测试 (#3) - 压力测试

## 输出格式测试建议：
# - 调试时使用 "output_format": "blob" 可直接下载查看
# - 或使用 "output_format": "file" 在服务器查看文件路径
# - 使用 "output_format": "zip" 如果blob格式有问题
# - 使用 "output_format": "base64" 用于工作流集成测试
